# 🍔 AHFAD Shehata - Restaurant Website

A modern, clean, and fully responsive restaurant website landing page built from scratch using HTML5 and CSS3.

---

## ✨ Features
* **Modern Hero Section:** Eye-catching welcome screen with professional typography.
* **Fully Responsive Design:** Optimized for all screen sizes including Desktops, Tablets, and Mobile devices (via CSS Media Queries).
* **Interactive Navigation:** Smooth scrolling behavior across sections (Home, About, Menu, Gallery, Contact).
* **Structured Sections:** Includes dedicated areas for special offers, food menus, image galleries, and a functional contact form.
* **Social Media Integration:** Built-in font-awesome icons linked directly to official social channels.
